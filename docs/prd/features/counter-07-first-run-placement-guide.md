# 子 PRD：首次摆放手机引导（开始）

| 优先级 | P1 |

| 预估 | 小 |

## 1. 目标

首次安装或首次进入「开始」时，用 **≤3 步** 图文说明手机摆放与距离，降低首次训练失败率。

## 2. 用户故事

作为新用户，我希望在第一次计数前知道手机应放哪里、人应站哪里。

## 3. 范围

**In**

- 持久化标记：`hasSeenPlacementGuide`（DataStore / SharedPreferences / Room 设置表任选一种，全 App 统一）。
- 未看过：在进入相机预览**之前**或**之后第一层**展示引导（Modal / 全屏向导）；**≤3 屏**，每屏：简图 + 一句说明 +「下一步」/ 末屏「知道了」。
- 看过不再展示；在「我的」提供「重新查看摆放说明」入口（可放在 P-04 或本需求末尾小字链接，二选一：本需求内做「我的」里一项跳转同一向导亦可）。

**Out**

- 视频教程、多语言全套（可先中文）。

## 4. 验收标准

1. 清除数据后引导再次出现；标记为已读后不再出现。
2. 引导期间不触发计数；「知道了」后进入与现有一致的「开始」页逻辑。
3. 横竖屏策略与主 PRD 一致：引导与训练使用**同一**屏幕方向策略（若训练仅竖屏，引导也仅竖屏）。

## 5. 内容建议（可替换为设计稿）

1. 将手机固定在腰部高度，画面能拍到全身。  
2. 距离约 2～3 米，避免背光。  
3. 点击开始后起跳，保持全身在画面内。

## 6. 手动测试要点

- 从「我的」重看引导（若实现）→ 返回「开始」不重置训练状态。
