# 子 PRD：历史空状态与去训练 CTA（历史）

| 字段 | 内容 |
|------|------|
| ID | H-04 |
| Tab | 历史 |
| 优先级 | P1 |
| 依赖 | H-01 |
| 预估 | 小 |

## 1. 目标

当无任何会话时，用友好空状态引导用户去「开始」完成第一次训练，形成**发现 → 行动**闭环。

## 2. 用户故事

作为新用户，打开历史看到空白时，我希望知道下一步该做什么。

## 3. 范围

**In**

- 列表数据为空时：展示插图或图标 + 标题 + 简短说明 + 主按钮「去跳绳」。
- 点击「去跳绳」：切换到 **开始 Tab**（与 `JumpMasterNavHost` 路由联动，可用 `NavController` 或共享事件，实现选最简单一种）。
- 有条目时不再显示空状态。

**Out**

- 运营位、广告。

## 4. 验收标准

1. 从「开始」保存第一条后回到历史，空状态消失并出现列表项。
2. CTA 导航 100% 到达「开始」页。
3. 深色模式可读。

## 5. 手动测试要点

- 删除全部记录（H-05 后）空状态再次出现（若 H-05 未做，用手动清 DB 测）。
