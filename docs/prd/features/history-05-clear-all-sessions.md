# 子 PRD：清空全部历史（历史）

## 1. 目标

提供**清空全部**会话能力，满足隐私与「重来」诉求；必须通过**二次确认**降低误操作。

## 2. 用户故事

作为用户，我希望一键清空所有历史，以便换账号或单纯想从零记录。

## 3. 范围

**In**

- 入口：历史页 Toolbar 菜单「清空全部」或「我的」里的危险区（二选一，**全 App 只保留一个入口**以免重复）。
- 第一次确认：说明后果；第二次确认：可输入「删除」或勾选「我了解不可恢复」（选一种轻量方案即可，不必过度）。
- 执行后：表清空或软删除（硬删除即可）；列表立即为空并显示 H-04 空状态。

**Out**

- 按条件筛选删除、云同步回滚。

## 4. 验收标准

1. 清空后 DB 会话数为 0（或等价无用户可见数据）。
2. 取消任意一步不产生删除。
3. 清空过程不 ANR（大数据量时仍用 IO 线程）。

## 5. 手动测试要点

- 清空后立刻开始新训练再保存，id 与列表正常。
