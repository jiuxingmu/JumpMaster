## 1. 目标

用户可关闭或开启**语音播报**（如里程碑报数），设置持久化并影响训练中的播报调用。

## 2. 用户故事

作为用户，我希望在安静环境关掉语音，只保留屏幕数字。

## 3. 范围

**In**

- 设置行「语音播报」+ 开关；持久化与 P-01 同方案。
- 所有 TTS 调用点在播报前检查开关；关闭时不初始化多余 TTS 或按需释放（避免泄漏，实现选型在 PR 说明）。
- 默认：`开` 或与现网一致。

**Out**

- 自定义播报内容、多语言音色。

## 4. 验收标准

1. 关 → 无语音；开 → 恢复（与现有里程碑逻辑对齐）。
2. 快速切换开关不崩溃、不排队播放旧任务（可 `cancel` 或忽略）。

## 5. 手动测试要点

- 系统 TTS 未安装引擎时的降级（Toast 或静默，**不崩溃**）。
